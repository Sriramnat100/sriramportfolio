# Sriram Portfolio

Next.js portfolio site. The app lives in **`updated-app/`**.

## Vercel

In Vercel project **Settings → General → Root Directory**, set:

```text
updated-app
```

Then redeploy. Builds run from that folder where `next` is in `package.json`.

## Local

```bash
cd updated-app
npm install
npm run dev
```

Open http://localhost:3000.
